The Serverless Workflow SDKs are essential tools designed to assist developers in consuming, parsing, validating, and testing their workflows utilizing the Serverless Workflow DSL.

These SDKs empower developers to seamlessly integrate serverless workflows into their applications, providing robust support for various programming languages. By offering comprehensive functionality, they streamline the development process and enhance workflow management.

## SDK Capabilities

All SDKs provide core functionality for working with Serverless Workflows:

### Reading and Writing

- Parse workflow definitions from JSON and YAML formats
- Serialize workflow objects back to JSON/YAML
- Support for both programmatic and file-based workflows

### Building

- Programmatic workflow construction using fluent APIs
- Builder patterns for creating complex workflow structures
- Type-safe workflow definition (in strongly-typed languages)

### Validation

- Validate workflow definitions against the specification
- Check for syntax errors and schema compliance
- Verify task references and data flow

### Testing

- Test workflow definitions before deployment
- Simulate workflow execution
- Validate input and output data structures

### Basic Usage Example

<CodeGroup>
```yaml Workflow Definition
document:
  dsl: 1.0.0
  namespace: examples
  name: hello-world
  version: 1.0.0
do:
  - sayHello:
      call: http
      with:
        method: post
        uri: https://api.example.com/greet
```

```typescript TypeScript SDK
import { Workflow } from '@serverlessworkflow/sdk-typescript';

// Parse workflow from YAML/JSON
const workflow = Workflow.fromFile('workflow.yaml');

// Validate the workflow
const errors = workflow.validate();

// Access workflow properties
console.log(workflow.document.name);
```

```python Python SDK
from serverlessworkflow import Workflow

# Load and parse workflow
workflow = Workflow.from_file('workflow.yaml')

# Validate the workflow
errors = workflow.validate()

# Access workflow properties
print(workflow.document.name)
```

```java Java SDK
import io.serverlessworkflow.api.Workflow;

// Parse workflow from file
Workflow workflow = Workflow.fromFile("workflow.yaml");

// Validate the workflow
List<ValidationError> errors = workflow.validate();

// Access workflow properties
System.out.println(workflow.getDocument().getName());
```
</CodeGroup>
